# Formation et Éducation

## EPITECH Rennes
**Programme** : Pre-MSc (Pre-Master of Science)  
**Niveau** : Bac+5 (équivalent Master)  
**Période** : 2025-2028
**Statut** : En cours

### Description
Formation d'excellence en informatique basée sur une pédagogie par projets innovante, où l'apprentissage se fait principalement par la pratique. Le programme Pre-MSc d'EPITECH met l'accent sur le développement de compétences techniques solides (développement web, architecture logicielle, DevOps) tout en développant l'autonomie, la créativité et l'esprit d'entreprise. Les étudiants travaillent sur des projets concrets et professionnalisants qui simulent des situations réelles du monde de l'entreprise.

### Domaines d'étude
- Développement web full-stack
- Architecture logicielle
- Algorithmes et structures de données
- DevOps et containerisation
- Gestion de projet agile

### Projets académiques
- Création d'un site de recherche d'emploi, avec utilisation de:
    - Next.js
    - Tailwind
    - Node.js
    - Express
    - PostgreSQL
- Création d'un jeu vidéo 2D, avec utilisation de:
    - JAVA
    - JavaFX
    - Maven

### Compétences développées
- Programmation Orientée Objet
- Frameworks modernes (Next.js, Node.js)
- Bases de données (PostgreSQL)
- Méthodologies agiles
- Travail en équipe
- Gestion versionning

## Formation OpenClassrooms
**Programme** : Développeur web fullstack 
**Période** : Janvier 2025-Septembre 2025  (10 mois)
**Statut** : Terminé

### Description
    - Front-end: conception d’interfaces responsives, intégration web, accessibilité et
    optimisation SEO (Google Analytics, Wave, Lighthouse), utilisation de
    frameworks modernes (React, TailwindCSS).
    - Back-end: développement d’API REST, gestion de DB MongoDB, sécurisation et
    optimisation (JWT, Bcrypt, Sharp), logique CRUD.
    Outils & méthodes: versioning Git/GitHub, Postman, Swagger, VSCode.

### Compétences acquises
- Développement front-end (React, TailwindCSS, HTML/CSS responsive)
- Développement back-end (Node.js, Express, API REST)
- Gestion de bases de données MongoDB (opérations CRUD)
- Sécurisation d'applications (JWT, Bcrypt)
- Optimisation et performance (Sharp, SEO, Lighthouse)
- Accessibilité web (WCAG, Wave)
- Versioning et collaboration (Git, GitHub)
- Tests et documentation API (Postman, Swagger)
- Google Analytics et suivi de performance
